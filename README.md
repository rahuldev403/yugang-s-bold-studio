# Yugang's Bold Studio

A modern, motion-first personal portfolio built with Vite, React, TypeScript, Tailwind CSS, and Framer Motion.

## Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Project Structure

- `src/components` - Main UI sections (`HeroSection`, `AboutSection`, `WorkSection`, `ContactSection`, etc.)
- `src/assets` - App assets used by components
- `public` - Static files (favicon, images, resume)

## Notes

- Favicon is served from `public/favicon.png`.
- Resume button in About links to `public/the_resume.pdf`.
- First selected project card links to the Rift prototype.
